Single-File Static Host

Drop this index.html into your web root (or serve from any static server). Includes quick host instructions and a live status check.

How to host this file

Recommended: put index.html in your web server's document root (example paths shown). Or run a one-liner local server.

Quick commands

# Place in nginx (Debian/Ubuntu)
sudo cp index.html /var/www/html/index.html
sudo chown www-data:www-data /var/www/html/index.html
sudo systemctl reload nginx

# Place in Apache2
sudo cp index.html /var/www/html/index.html
sudo chown www-data:www-data /var/www/html/index.html
sudo systemctl reload apache2

# Quick local test (no root required)
python3 -m http.server 8080
# then open: http://localhost:8080

# Node (if you have npm):
npx serve -s .  # or: npx http-server -p 8080

Notes

  • If both nginx and apache run on same machine, ensure different ports (e.g. apache → 8080).
  • Use HTTPS for production (Let’s Encrypt certs or your preferred method).
  • To host from GitHub: push this to a repo and enable GitHub Pages (branch: main / docs folder).
